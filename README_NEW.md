# Sehyun Kwon's Personal Website

This is the personal academic website of Sehyun Kwon, built in the style of [Jon Barron's website](https://jonbarron.info/).

## Viewing the Website

Simply open `index.html` in your web browser to view the website locally.

To publish on GitHub Pages:
1. Push this repository to GitHub
2. Go to repository Settings > Pages
3. Select "Deploy from a branch" and choose the main branch
4. Your site will be available at `https://sehyunkwon.github.io`

## Structure

- `index.html` - Main website page
- `stylesheet.css` - Styling for the website
- `assets/img/` - Images directory
  - `IMG_2365.JPG` - Profile photo
  - `default-paper.png` - Default paper thumbnail
